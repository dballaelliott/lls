# Print method for lls objects

Print method for lls objects

Summary method for lls objects just an alias for print

## Usage

``` r
# S3 method for class 'lls'
print(lls.return, ...)

# S3 method for class 'lls'
summary(lls.return, ...)
```

## Arguments

- lls.return:

  An lls object

- ...:

  Additional arguments (ignored)

- object:

  An lls object
