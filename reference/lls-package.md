# lls: Local Least Squares Estimation for Information Provision Experiments

Implements the Local Least Squares (LLS) estimator to recover Average
Partial Effects (APE) in information provision experiments, robust to
dependence between belief updating and belief effects. Provides plotting
helpers and example data.
